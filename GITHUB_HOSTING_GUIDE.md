# How to Host This Portfolio on GitHub Pages

## Option A: Host as your main portfolio site

1. Go to GitHub and create a new public repository named:

   `briankodibona.github.io`

2. Upload the contents of this folder, not the folder itself. Your repository should look like this:

   ```text
   index.html
   README.md
   GITHUB_HOSTING_GUIDE.md
   assets/profile.jpg
   ```

3. Commit the files.

4. Wait 1 to 3 minutes. Your website should be live at:

   `https://briankodibona.github.io`

## Option B: Host inside any repository

1. Create a repository, for example:

   `ml-portfolio-showcase`

2. Upload all files from this folder.

3. Go to **Settings** → **Pages**.

4. Under **Build and deployment**, choose:

   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`

5. Click **Save**.

Your site will be published at a GitHub Pages link.
