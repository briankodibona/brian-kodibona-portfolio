Showcase submission plan

1. Choose one flagship project

2. Clean the GitHub repository

Add README, notebook, requirements file, images, and a one-paragraph business summary.

3. Add a visual demo

4. Write the case study

Structure it as: context, data, method, result, business impact, limitations, next steps.
